### component的sass

component 即使使用sass编写 也无法直接使用 样式代码很可能发生重复

shadow dom 还不太成熟 那么bem作为过渡方案是不错的

一个component需要一份.d.scss的文件

```
test-element.h.scss
@component test-element {
    some property
    @block .test-element {
        some property
        @modify default {
            some property
            @elment up {
              some property
            }
        }
    }
}

// 这时候test-element 并没有生成css

// 当然你也可以选择不去使用test-element的css
// 因为只是一份对象数据 所以可以继承
@component your-test-element @extends test-element {
    @super;
    @block .test-element {
        // 你可以指定override
        @override @modify default {
        }        
    }    
}

// 调用register才会生成 css
// your stylesheet scss
@include test-element.h;

@register-component test-element;
```

要实现这些首先的扩充sass的map

```
@function map-deep-get($map, $keys...) {
  @each $key in $keys {
    $map: map-get($map, $key);
  }

  @return $map;
}

@function map-deep-set($map, $keys.../*, $value */) {
  $map-list: ($map,);
  $result: null;

  @if length($keys) == 2 {
    @return map-merge($map, (nth($keys, 1): nth($keys, -1)));
  }

  @for $i from 1 through length($keys) - 2 {
    $map-list: append($map-list, map-get(nth($map-list, -1), nth($keys, $i)));
  }

  @for $i from length($map-list) through 1 {
    $result: map-merge(nth($map-list, $i), (nth($keys, $i): if($i == length($map-list), nth($keys, -1), $result)));
  }

  @return $result;
}


@function map-has-keys($map, $keys...) {
    @each $key in $keys {
        @if not map-has-key($map, $key) {
            @return false;
        }
    }

    @return true;
}

@function map-has-nested-keys($map, $keys...) {
    @each $key in $keys {
        @if not map-has-key($map, $key) {
            @return false;
        }
        $map: map-get($map, $key);
    }

    @return true;
}

@function map-get-key($map, $index) {
    @return nth(map-keys($map), $index);
}
```
